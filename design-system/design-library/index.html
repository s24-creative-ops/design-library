Library-Prinzip

Diese Veroeffentlichung bildet das Arbeitsprinzip bewusst sichtbar ab.

Tokens werden direkt aus dem Brand-JSON geladen. Die Modul-Bereiche spiegeln die aktiven Kataloge von Email- und LP-Builder. Damit bleibt die Library in ihrer Struktur auf die eigentlichen Source-Artefakte ausgerichtet und nicht auf eine separate, manuell gepflegte Parallelwelt.
