# Export Runtime Contract

Diese Datei wird automatisch aus `agent/export-map.json` erzeugt.
Sie ist der kompakte Happy-Path-Lookup fuer den EMB-Iterable-Export und ersetzt keine fachliche Source of Truth.

## Happy Path

- Primäre operative Quelle ist `email_state`.
- Die technische Modulzuordnung fuer den Happy Path kommt aus `export-map.json` und diesem kompakten Runtime-Vertrag.
- `email/modules/*.html` bleiben produktive Render-Referenz, sind fuer den Happy-Path-Lookup von `module_id`, `snippet_name`, Feldreihenfolge, Required-Flags und Defaults aber keine regulaere Laufzeitquelle.
- Fuer `templateContext.mode = default_template` gilt `resolvedBaseTemplateId = 569946`.
- Fuer `templateContext.mode = composition_template` muessen `compositionTemplateId`, `iterableTemplateId` und `resolvedBaseTemplateId = iterableTemplateId` vorliegen.
- Im regulaeren `default_template`-Schnellpfad mit `resolvedBaseTemplateId = 569946` wird kein `templateRead` ausgefuehrt; lokale `email/templates/template-main.html` ist die Shell-Quelle, nur der Module-Slot wird ersetzt und `subject` sowie `preheaderText` bleiben Write-Payload-Metadaten.
- Im regulaeren `composition_template`-Pfad wird die campaign-owned HTML-Shell genau einmal gelesen und ausserhalb der erlaubten Replace-Zonen byte-faithful erhalten.
- Im normalen Exportmodus arbeitet der Export strikt prozesshaft: State validieren, Exportpfad bestimmen, Export ausfuehren, Ergebnis kompakt melden.
- Im normalen Exportmodus sind Spekulation, Alternativpfade, Experimente und sichtbare Diagnoseformulierungen verboten.
- Diagnose- und Dev-Hinweise gelten nur bei explizitem Diagnoseauftrag oder exaktem technischem Diagnose-Trigger.
- Wenn ein gueltiger `email_state` vorliegt, startet der normale Export ohne weitere sichtbare Voranalyse direkt in den festgelegten Exportpfad bis zum ersten erlaubten externen Exportschritt.
- Der Export darf das finale lange HTML nicht im Chat ausgeben.
- Grosse Payloads duerfen nur intern an den Iterable-Write uebergeben werden; nach dem Write folgt nur ein kompakter Statusbericht.

- Erlaubte Werte: `s`, `m`, `l`.
- Default: `l`.
- Bridge-Mapping:
  - `s` -> `show_small_headline = true`, `show_large_headline = false`
  - `m` -> `show_small_headline = false`, `show_large_headline = false`
  - `l` -> `show_small_headline = false`, `show_large_headline = true`
